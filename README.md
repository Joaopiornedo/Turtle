# Controle de Movimento da Tartaruga
Projeto em Python que usa o módulo turtle para controlar
a movimentação de uma tartaruga na tela de forma interativa.


### Instruções
Escolha a direção:

'f' para frente
't' para trás
Informe a distância.

### Escolha a rotação:

'e' para esquerda
'd' para direita
'n' para não rotacionar

#### Escolha se deseja continuar:

's' para sim
'n' para não
